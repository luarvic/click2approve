Click2Approve - Document Approval System

What is Click2Approve

Click2Approve is a free, open-source, cross-platform document approval system with a responsive user interface that allows you to:

  • Upload documents.
  • Send documents for approval specifying a list of approvers' email addresses.
  • Notify the requesting and approving parties via email.
  • Keep track of the approval requests.

How to use the document approval system

You can start using the document approval system right away by clicking Click2Approve.com/ui.

Or you can host it on your servers and use the README file as a starting point for the setup details.

If you have any questions or need help, please start a discussion in the GitHub Discussions.
